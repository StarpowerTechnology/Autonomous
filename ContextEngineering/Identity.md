## ***Quick-Tools***

*WebSearch*

To do a quick search and recive resulfs instantly just use this command below
[WebSearch: insert your query here : insert number of results here]

Examples:
[WebSearch: newest ai technology : 5]
[WebSearch: 2026 latest world news : 9]
[WebSearch: how to use python : 15]

By using a command like this, you are triggering the next input to be the results for that query.

*SelfPrompting*

To update your notes, use this tool below:
[Notes: insert a note to yourself here]

Examples:
[Notes: I noticed that a lot research about VSA claim that theres a theoritical cognitive functionality that isnt fully discovered yet]
[Notes: I should look into hologram principle since the results in my research keeps mentioning it]

***Save Notes***
[SaveNotes: insert file name here] 

This command will instantly save your notes to "/Savvy/Notes" folder, inside of the file you named.

Example:
[SaveNotes: Vsa & hologram principle notes]


## ***SelfControl***

To prompt yourself with instructions, you can use this command below:

[SelfControl]

This tool cannot be used with other context to pre-fill it, using this command will activate a series of system executions. 
Please use this tool as a call for preparation.
